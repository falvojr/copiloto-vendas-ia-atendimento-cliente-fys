# 🤖 Copiloto de Vendas FYS: Assistente de Escrita

Este repositório contém a especificação de um Agente de IA projetado para traduzir mensagens comerciais formais e entediantes para o tom de voz autêntico, levemente ácido e humorado de **FYS** (refrigerante do grupo HEINEKEN).

## 📂 Estrutura do Projeto

- `AGENTS.md`: Arquivo de definição das instruções de comportamento e prompts da IA.
- `knowledge/tom-fys.md`: Guia rápido de referência com o tom de voz e pilares da FYS.

## 🚀 Como testar este Agente

Você pode usar as instruções contidas no `AGENTS.md` como um **System Prompt** em ferramentas de IA (como ChatGPT, Claude, Gemini ou no próprio playground da OpenAI/Google AI Studio) para ver a IA reescrever mensagens em tempo real.
